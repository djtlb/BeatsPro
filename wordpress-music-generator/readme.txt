AI-powered music generation plugin for WordPress

== Description ==

Beat Addicts Music Generator allows you to create AI-generated music directly from your WordPress site. Users can input prompts, select genres and moods, and generate custom music tracks.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/beat-addicts-music-generator/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the shortcode `[beat_generator]` in any post or page

== Usage ==

Add the shortcode `[beat_generator]` to any post or page where you want the music generator to appear.
